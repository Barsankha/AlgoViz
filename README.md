# AlgoViz 🚀

A modern Flutter application for visualizing algorithms and data structures interactively.

---

## ✨ Features

- 🎨 Beautiful modern UI
- 📊 Algorithm visualization
- 🔄 Sorting animations
- 🌲 Data structure demonstrations
- ⚡ Smooth Flutter animations
- 📱 Cross-platform support
